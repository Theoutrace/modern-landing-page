# Modern Landing Page

A beautiful, responsive landing page with three main sections:
- **Hero Section**: Eye-catching introduction with call-to-action
- **Features Section**: Showcase of key features and benefits
- **About Section**: Information about the project with statistics

## Features

- Modern, gradient-based design
- Fully responsive layout
- Smooth scrolling navigation
- Hover effects and animations
- Clean, semantic HTML5
- CSS3 animations and transitions

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript

## Getting Started

Simply open `index.html` in your web browser to view the landing page.

## Deployment

This project is automatically deployed to GitHub Pages using CI/CD.

### Live Site

The site is deployed at: `https://theoutrace.github.io/modern-landing-page/`

### CI/CD Setup

The project uses GitHub Actions for continuous deployment:
- **Workflow**: `.github/workflows/deploy.yml`
- **Trigger**: Automatically deploys on every push to the `main` branch
- **Platform**: GitHub Pages

### Manual Deployment

To manually trigger a deployment:
1. Go to the Actions tab in GitHub
2. Select "Deploy to GitHub Pages" workflow
3. Click "Run workflow"

### Enabling GitHub Pages

If GitHub Pages is not already enabled:
1. Go to repository Settings â†’ Pages
2. Under "Source", select "GitHub Actions"
3. The workflow will automatically deploy on the next push
